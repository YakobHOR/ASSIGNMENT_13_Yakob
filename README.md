# ASSIGNMENT_13_Yakob
This project demonstrates the fundamentals of Generative AI by building a simple character‑level text generation model using TensorFlow/Keras. The model is trained on a public‑domain dataset from Project Gutenberg and generates new text based on a seed input. The project also includes an explanation of GPT architecture, ethical considerations, and a practical content‑creation application.

This project uses a public‑domain text from Project Gutenberg:

Alice’s Adventures in Wonderland

URL: https://www.gutenberg.org/cache/epub/11/pg11.txt (gutenberg.org in Bing)

The dataset is downloaded directly in the Colab notebook.
